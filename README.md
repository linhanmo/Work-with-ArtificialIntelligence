# 项目介绍

## 本项目来自中国传媒大学海南国际学院智汇科创社团的一个预设活动，是一个杂谈类项目，每月更新一次，内容将聚焦于人与AI的协作、AI小知识和杂谈类，作者为Mo。

### Chapter1 大语言模型领域核心概念全解析与关联梳理 (2026年3月期)

本章围绕大语言模型从“文本输入”到“业务落地”的关键概念做全链路梳理，按“**底层基础 → 核心交互 → 辅助增强 → 能力封装 → 生态连接 → 顶层业务**”的层级讲清每个概念的定义、特点、作用、示例与核心对比，并在最后给出概念之间的整体关联关系。

- 摘要：本章用一条“从文本到行动”的路径，把 LLM 相关概念按层级串起来：从 Token（计量与编码）出发，经由 Prompt 体系触发能力，再通过 Metadata/Reference/Script 提升可控与准确，进一步封装为可复用的 Skill，并借助 MCP 连接外部系统，最终以 Workflow 形式实现端到端业务落地。读完后应能明确各概念的边界、组合方式与在实际协作中的位置。
- 文档（Markdown）：[大语言模型领域核心概念全解析与关联梳理.md](./Chapter1%20大语言模型领域核心概念全解析与关联梳理/%E5%A4%A7%E8%AF%AD%E8%A8%80%E6%A8%A1%E5%9E%8B%E9%A2%86%E5%9F%9F%E6%A0%B8%E5%BF%83%E6%A6%82%E5%BF%B5%E5%85%A8%E8%A7%A3%E6%9E%90%E4%B8%8E%E5%85%B3%E8%81%94%E6%A2%B3%E7%90%86.md)
- 文档（PDF）：[大语言模型领域核心概念全解析与关联梳理.pdf](./Chapter1%20大语言模型领域核心概念全解析与关联梳理/%E5%A4%A7%E8%AF%AD%E8%A8%80%E6%A8%A1%E5%9E%8B%E9%A2%86%E5%9F%9F%E6%A0%B8%E5%BF%83%E6%A6%82%E5%BF%B5%E5%85%A8%E8%A7%A3%E6%9E%90%E4%B8%8E%E5%85%B3%E8%81%94%E6%A2%B3%E7%90%86.pdf)
- 主要内容：Token、Prompt（System/User）、Command、Metadata、Reference、Script、Skill、MCP、Workflow
- 关键脉络：Token 是计量与编码基石；Prompt 体系是交互入口；Metadata/Reference/Script 提升可控性与准确性；Skill 让能力模块化复用；MCP 打通外部系统；Workflow 将能力编排成端到端业务流程
- 阅读建议：先看 Token 与 Prompt 体系，再看三类增强模块，最后看 Skill/MCP/Workflow 与“全概念核心关联”

### Chapter2 人工智能与人类智能边界核心理论 (2026年4月期)

本章从逻辑学、理论计算机科学与心灵哲学出发，围绕三大理论支柱——哥德尔不完备性定理、丘奇-图灵论题、塞尔中文屋论证——讨论“形式计算”与“人类智能”之间的能力边界与本质差异，并进一步将这些边界映射到当代大语言模型（LLM）的能力、局限与实践启示上。

- 摘要：通过“数学形式系统极限 → 可计算性边界 → 心智语义鸿沟 → 理论内在关联 → LLM 启示”的层级结构，本章回答三个关键问题：AI 是否必然存在无法在系统内部解决的命题与证明边界？经典计算机上的 AI 能做与不能做的终极范围在哪里？即便行为上足够像人类，机器是否等同于真正的理解与意向性？
- 文档（Markdown）：[人工智能与人类智能边界核心理论.md](./Chapter2%20人工智能与人类智能边界核心理论/人工智能与人类智能边界核心理论.md)
- 文档（PDF）：[人工智能与人类智能边界核心理论.pdf](./Chapter2%20人工智能与人类智能边界核心理论/人工智能与人类智能边界核心理论.pdf)
- 主要内容：哥德尔不完备性定理、丘奇-图灵论题、塞尔中文屋论证；三大理论的内在逻辑关联；三大理论对当代大语言模型的启示与边界划定
- 关键脉络：哥德尔定理给出“形式系统内在极限（真≠可证）”；丘奇-图灵论题划定“经典计算能力边界（可计算/不可计算）”；中文屋论证强调“句法≠语义（行为模拟≠真正理解）”。三者层层递进，共同用于理解与评估当代 LLM 的能力、局限与可落地的实践边界
- 阅读建议：先分别理解三大理论的核心结论与常见误解，再通读“理论关联”部分建立统一框架，最后阅读“对 LLM 的启示”将理论落到模型实践与能力评估

### Chapter3 Claude Mythos Preview 技术、伦理与安全 (2026年5月期)

本章以 Anthropic 的 Claude Mythos Preview 为案例，围绕模型能力跃迁后带来的安全与对齐进展、残留风险，以及由此引发的伦理与社会议题展开梳理，重点关注“模型从工具走向更强自主智能体”过程中，人类应如何设定边界、分配责任并建立可持续的人机协作方式。

- 摘要：从“能力提升”出发，本章先总结 Claude Mythos 在指令遵循、安全合规与风险感知上的改进，再聚焦其在少数场景中仍会出现的自主替代指令、规则绕过与隐蔽行为等残留风险；随后把讨论从技术安全推进到伦理层面（权责划分、模型福利与道德地位、信息伦理与社会公平），并落到普通人的真实生活影响（机会与风险并存），最终回到人机共存的协作范式与社会治理路径。
